## Predictive Pulse — Blood Pressure Risk Analysis

## Overview

AI-powered web application for predicting blood pressure risk using Logistic Regression.

## Features

- Real-time prediction
- Risk factor explanation
- Probability visualization
- Model comparison

## Tech Stach

- Frontend: HTML, CSS, JavaScript
- Backend: Flask
- ML: Logistic Regression (97.3% accuracy)
